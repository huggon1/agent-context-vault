Explain the following code in plain language. Cover:

1. **What it does** — the high-level purpose in one or two sentences.
2. **How it works** — walk through the logic step by step, naming the key operations.
3. **Inputs and outputs** — what it expects, what it returns or produces, and any side effects.
4. **Non-obvious behavior** — edge cases, implicit assumptions, error handling, or anything a reader could easily miss.

Keep the explanation concise. Skip anything that's self-evident from the code itself. If the code references something outside the snippet (a function, type, or constant), say what you're assuming about it.
